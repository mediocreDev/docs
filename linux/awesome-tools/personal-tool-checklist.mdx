---
title: "Personal Tool Checklist"
description: "My list of the Must-install tools for a fresh Linux PC."
---

- [btop](https://github.com/aristocratos/btop) - Better `top`/`htop`

- [mosh](https://mosh.org/#getting) - Roaming Shell

- [most](https://queirozf.com/entries/linux-most-usage-examples-for-a-little-known-pager-tool) - Better pager than `less`

- [zsh](/linux/awesome-tools/system/zsh)/OhMyZsh/OhMyPosh/Powerlevel10k - Pretty Shell combination

- [eza](/linux/awesome-tools/applications/others#eza)/[lsd](/linux/awesome-tools/applications/others#lsd) - Prettier `ls`

- [bat](https://github.com/sharkdp/bat) - Better `cat`

- [posting.sh](https://posting.sh/guide/) - Terminal Postman

- [curl](https://justuse.org/curl/) - Built-in (or not?)
