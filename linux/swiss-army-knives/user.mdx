#### List users

```shell
# List all users
cat /etc/passwd
cut -d: -f1 /etc/passwd
who
users
# List system users
awk -F':' '$3 < 1000 {print $1}' /etc/passwd
# List normal users
awk -F':' '$3 >= 1000 {print $1}' /etc/passwd
# List all users
getent passwd
# List all groups
getent group
```

#### Create user

```shell
sudo adduser americio
#sudo mkhomedir_helper americio
sudo usermod -aG sudo americio
#sudo passwd americio
su americio
```

#### Reset password

```shell
sudo passwd $USER_NAME
```

#### User Information

```shell
id $USERNAME
# Current logged-in user
id
# Detailed User Info (from /etc/passwd)
getent passwd $USERNAME
# User's groups
groups username
```

### Rename group:

```shell
sudo groupmod -n $NEW_GROUP_NAME $OLD_GROUP_NAME
```
